# Trueview Strategic Lead Portal 🚀
### Intelligent B2B Lead Generation MVP for Vilican AI Lab Assignment

A professional-grade AI-powered lead generation tool designed for **Trueview Warner Pvt. Ltd.** This application automates the process of finding high-value procurement leads for security systems, interactive displays, and water quality solutions.

## 🔗 Live Links
- **Live Demo:** https://trueview-lead-generator.vercel.app/
- **Source Code:** (https://github.com/AshuNimbalkar-02/Trueview-Lead-Generator)

## ✨ Key Features
- **Intelligent Lead Scouting:** Targeted searches for CCTV, Interactive Panels, Video Walls, and TDS Meters.
- **Dual Engine Modes:**
  - **Cached Region Mode:** High-speed retrieval from optimized regional databases.
  - **Real-Time Mode:** Simulated live scouting via public API integration to demonstrate network handling.
- **Smart Shortlisting:** Persistent bookmarking system to save high-priority leads for follow-up.
- **Advanced Data Export:** Professional export options for **PDF, Excel (XLSX), CSV, and JSON**.
- **AI Sales Strategist:** Built-in dashboard providing procurement readiness scores and sales recommendations.
- **Premium UI/UX:** Responsive design featuring glassmorphism, smooth Framer Motion animations, and custom branding.

## 🛠️ Technology Stack
- **Core:** React.js + Vite
- **Styling:** Vanilla CSS (Custom UI System)
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **Data Export:** jsPDF, SheetJS (XLSX)

## 🚀 Getting Started
1. Clone the repository
2. Run `npm install`
3. Run `npm run dev` to start the local development server

## 📁 Project Structure
- `src/App.jsx`: Core application logic and state management.
- `src/App.css`: Professional design system and animations.
- `src/assets/`: Branding and static resources.
